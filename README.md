# setup our Dapp

Try running some of the following tasks to get started
```shell
npm install
```
## To start our contract
compile the contract
```shell
npx hardhat compile
```

start our local node
```shell
npx hardhat node
```

deploy contract
```shell
npm run contract
```

Start HTTP server
```shell
npm run server
```

Navigate to `localhost:8080/frontend/index.html`
